How the Internet Works

The internet is a system designed to move information. The internet ships binary information known as bits. One bit has two possible values 0 and 1 or on and off. 8 bits strung together result in something called a byte. (8 bits = 1 byte). The internet is designed in a way that if you use the same protocols communication from one device to another should be very easy.
